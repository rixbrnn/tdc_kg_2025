# Passo do Conversor (RDB → RDF)

## Instalar dependências

```bash
npm install
```

## Executar script de Mapeamento
```bash
npm start
```

A saída será localizada em:
`../rdf/output.trig`

## Executar Testes
```bash
npm test
```
- Testes aprovados confirmam que os mapeamentos implementados estão corretos.
- Testes falhando indicam TODOs em `src/mapping_rdf_todo.js` que precisam ser completados.
