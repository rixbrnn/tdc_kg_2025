This is a bonus part to visualize the ontologies and knowledge graphs through a web-based application:
Note: For this to work you have to ensure you have the blazegraph instance running on port 8889 and with the rdf data already uploaded there.
1. Clone [https://github.com/zazuko/graph-explorer.git](https://github.com/rixbrnn/graph-explorer-blazegraph-demo)
2. Run npm install
3. Run npm run build
4. Run npm run demo
5. Access http://localhost:10444/
