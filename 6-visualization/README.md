Esta é uma parte bônus para visualizar as ontologias e grafos de conhecimento através de uma aplicação web:
Nota: Para isso funcionar, você precisa garantir que tem a instância do blazegraph rodando na porta 8889 e com os dados rdf já carregados lá.
1. Clone [https://github.com/zazuko/graph-explorer.git](https://github.com/rixbrnn/graph-explorer-blazegraph-demo)
2. Execute npm install
3. Execute npm run build
4. Execute npm run demo
5. Acesse http://localhost:10444/

3. Run npm run build
4. Run npm run demo
